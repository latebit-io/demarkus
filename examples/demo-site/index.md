# Welcome to Demarkus

This is an example markdown site served over the Mark Protocol.

## What is Demarkus?

Demarkus is a privacy-first, markdown-native web protocol. It reimagines the web around documents, with:

- **Privacy by default** - No tracking, no cookies, no surveillance
- **Markdown-native** - Content is just markdown files
- **Distributed** - Every reader caches content, enabling preservation
- **Simple** - Human-readable protocol, minimal complexity

## Protocol Features

1. **Clean URLs**: `mark://example.com/doc.md`
2. **Versioning**: `mark://example.com/doc.md/v41`
3. **No query strings**: Parameters via frontmatter
4. **Text-based status**: `ok`, `not-found`, not HTTP codes

## Try These Commands

In the Demarkus client, you can:

- Press `g` to enter a URL
- Press `f` to follow a link
- Press `h` to go back in history
- Press `l` to go forward
- Press `q` to quit
